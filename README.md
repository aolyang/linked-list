# linked-list

+ Vanilla Typescript **DoublyLinkedList** library.  
+ **Array Similarity** experience.  
+ **Iterator** support.
+ **Rich Api** interface.
+ **Test** keep it stable.

## api

- [x] insert
- [x] forEach
- [x] map
- [x] filter
- [x] find
- [x] findIndex
- [x] push
- [x] remove
- [x] shift
- [x] unshift
- [ ] more api, types...

## Simple usages

```typescript
import { LinkedList } from ""

const myList = new LinkedList([1, 2, 3])

```
